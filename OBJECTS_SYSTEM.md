# Система управления объектами недвижимости

## Обзор

Реализован полнофункциональный раздел "Объекты" с фильтрацией, детальными страницами и системой избранного.

## Структура проекта

```
src/
├── components/
│   ├── objects/
│   │   ├── ObjectsPage.tsx          # Каталог с фильтрами и сеткой объектов
│   │   ├── ObjectCard.tsx           # Карточка объекта для каталога
│   │   ├── ObjectsFilters.tsx       # Панель фильтров
│   │   └── ObjectDetailPage.tsx     # Детальная страница объекта
│   ├── Header.tsx                   # Хедер с навигацией и избранным
│   └── HomePage.tsx                 # Главная страница с призывом к действию
├── types/
│   └── investment-object.ts         # Типы для объектов и фильтров
└── utils/
    └── initMockObjects.ts           # Инициализация демо-данных
```

## Маршруты

- `/` - Главная страница
- `/objects` - Каталог объектов с фильтрацией
- `/objects/:id` - Детальная страница объекта

## Функциональность

### 1. Каталог объектов (/objects)

**Фильтрация:**
- Поиск по названию/адресу
- Город (мультиселект)
- Тип недвижимости (квартиры, апартаменты, коммерческая, загородная)
- Бюджет (слайдер от 0 до 500 млн)
- Доходность (до 30%, 31-50%, 51-100%, свыше 100%)
- Срок окупаемости (до 5 лет, 5-8, 8-12, свыше 12)
- Статус (свободен, бронь, продано)

**Сортировка:**
- По умолчанию
- По доходности (↑↓)
- По цене (↑↓)
- По дате добавления

**Особенности:**
- Мгновенное обновление результатов
- Сохранение фильтров в localStorage
- Адаптивный дизайн (мобильные фильтры)
- Счетчик найденных объектов

### 2. Карточка объекта

**Отображает:**
- Фото объекта
- Название и адрес
- Тип недвижимости и площадь
- Доходность (% годовых)
- Сумма входа (млн ₽)
- Срок окупаемости (лет)
- Статус (бейдж)
- Кнопка "Избранное"

**Действия:**
- Клик по карточке → переход на детальную страницу
- Клик по сердечку → добавить/убрать из избранного

### 3. Детальная страница объекта

**Разделы:**
- Галерея фото (с навигацией)
- Основная информация (название, адрес, статус)
- Ключевые показатели (доходность, сумма входа, окупаемость)
- Описание объекта
- Расчет доходности (таблица по месяцам на год)
- Информация о брокере (имя, компания, рейтинг, контакты)
- Документы для скачивания
- Форма заявки на консультацию

### 4. Система избранного

- Добавление/удаление объектов в избранное
- Сохранение в localStorage
- Счетчик в хедере (показывается только если есть избранные)
- Иконка сердца на карточках (красная = в избранном)

## Хранение данных

### localStorage ключи:

```javascript
'investment-objects'      // Массив всех объектов
'object-filters'          // Состояние фильтров
'favorites'               // Массив ID избранных объектов
'investpro-user'          // Данные пользователя
```

### Структура объекта:

```typescript
interface InvestmentObject {
  id: number;
  title: string;
  type: 'apartments' | 'flats' | 'commercial' | 'country';
  city: string;
  address: string;
  price: number;
  yield: number;
  paybackPeriod: number;
  area: number;
  status: 'available' | 'reserved' | 'sold';
  images: string[];
  description: string;
  brokerId: number;
  createdAt: string;
  monthlyIncome?: number;
  rentalYield?: number;
  documents?: { title: string; url: string }[];
  coordinates?: { lat: number; lng: number };
}
```

## Интеграция с кабинетом брокера

Брокеры могут управлять объектами через свой кабинет (вкладка "Объекты каталога"):
- Добавить новый объект
- Изменить статус (свободен/бронь/продано)
- Посмотреть объект на сайте
- Удалить объект

Все изменения мгновенно отражаются в каталоге.
